## Introduction

Neo Voyage is a responsive landing page application built for an exchange agency focused on travel experiences in Canada. The project presents program information, destination content, and service details through a mobile-friendly interface with animated transitions and structured navigation.

The application is designed for users searching for an exchange agency website, travel agency landing page, study abroad platform, or responsive tourism interface built with Bootstrap and JavaScript.

## Technical Overview

The project is built using HTML5, JavaScript, and Bootstrap for responsive layout structure and UI components. Animations are implemented with the AOS (Animate On Scroll) library to improve section transitions and visual interaction during navigation.

Fancybox is used for media presentation and gallery interactions. The layout follows a section-based architecture optimized for marketing pages, service descriptions, and travel program presentation.

The frontend is fully static and can be deployed through platforms such as Vercel, Netlify, or GitHub Pages without backend dependencies.

## Developer Instructions

### Project structure

- `index.html` - main application page
- `assets/` - static assets and images
- `css/` - custom styles
- `js/` - JavaScript interactions and animations

### Development workflow

- Maintain responsive behavior across breakpoints
- Keep animation usage lightweight and accessible
- Optimize image sizes before deployment
- Preserve semantic HTML structure for SEO and accessibility

### Contribution guidelines

- Create a feature branch from `main`
- Keep pull requests focused on one change
- Use descriptive commit messages
- Test responsiveness before opening a PR
- Include screenshots for UI modifications

Suggested branch names:

```bash
feature/program-section
fix/mobile-navbar
chore/readme-update
```

## Known Issues

- No CMS or dynamic content management
- Contact forms are static unless connected to an external backend service
- Animations may require optimization on lower-end mobile devices
